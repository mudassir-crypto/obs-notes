Bulk inserts:
```sql
INSERT INTO employees (first_name, last_name, age, department)
VALUES 
('John', 'Doe', 30, 'HR'),
('Jane', 'Smith', 25, 'Finance'),
('Emily', 'Jones', 35, 'IT');

```

Updating multiple columns:
```sql
UPDATE table_name
SET column1 = value1, 
    column2 = value2, 
    column3 = value3
WHERE condition;

```
